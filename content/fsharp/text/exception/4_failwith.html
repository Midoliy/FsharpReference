failwith関数の概要


　failwith関数は、F#例外を発生させるために利用されます。
// ------------------
// [ 構文 ]
// ------------------
failwith error-message-string

　raise関数は自作のF#例外を発生させるために利用されますが、failwith関数はデフォルトのF#例外を発生させることができます。[error-message-string] には文字列リテラルかstring型の値を指定することができます。また、指定した値はwith句内で束縛させることが可能です。
　failwith関数によって発生した例外は、System.Exception型と同等です。また、この例外をパターンマッチするために、Failure がよく利用されます。
