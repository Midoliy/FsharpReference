型略称の概要


　型略称は、コードを読みやすくするために型の省略系を宣言してわかりやすい名前にする機能のことを言います。特に長いクラス名や構造体名を短く記述したい場合に便利です。

　以下は型略称の構文になります。
// ------------------
// [ 構文 ]
// ------------------
type type-abbreviation = type-name

　構文を見てもらうとわかるとおり、非常に単純な構造をしていることがわかります。
　使い方も非常に簡単です。
