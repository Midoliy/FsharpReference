while...do式の概要


　while...do式は、指定されたテスト条件式が真(= true)である限り繰り返しをし続けるための構文です。for...to式と同様、命令型言語ではポピュラーなループ構文です。
// ------------------
// [ 構文 ]
// ------------------

while test-expression do
    body-expression

　まず最初にテスト式(= test-expression)が評価されます。その評価結果が真の場合にのみ実行式(= body-expression)が実行され、再度テスト式が再度評価されます。実行式はunit型である必要があります。
